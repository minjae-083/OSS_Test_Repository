# Todo Project

간단한 할 일 관리 프로그램입니다.

## 기능

- 할 일 추가
- 할 일 삭제
- 할 일 목록 조회
- 할 일 완료 처리

## 프로젝트 구조

```
todo-project/
├── src/
│   ├── todo.py       # 핵심 기능 (추가, 삭제, 조회)
│   └── utils.py      # 유틸 함수 (날짜 포맷 등)
├── tests/
│   └── test_todo.py  # 테스트 코드
└── .gitignore
```

## 실행 방법

```bash
cd todo-project/src
python todo.py
```
